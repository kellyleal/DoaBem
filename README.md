# DoaBem

Este projeto foi desenvolvido durante o Hackathon da CubosAcademy e Semcomp.

## App BemDoa

O App BemDoa é uma plataforma que conecta empresas interessadas em doar alimentos. O aplicativo permite o cadastro de empresas e a recepção de doações.

![image](https://github.com/kellyleal/DoaBem/assets/85024687/f45f3213-0bf0-46ec-960f-020c95596e44)

## Sobre o Projeto

Doa Bem é uma iniciativa que apoia as ações para alcançar o objetivo de Fome Zero e promover a Agricultura Sustentável, um dos 17 desafios definidos pela ONU. Convidamos você a se juntar a nós nessa causa.
